Pencasts
Detta är pencasts, dvs animerade pdf-dokument med ljud. Man kan öppna dem i en vanlig pdf-läsare men då blir de inte animerade och utan ljud (ganska värdelösa). Men om man installerar gratisprogrammet Echo Desktop (finns nedladdningsbart från https://www.livescribe.com/site/livescribe-2/echo för Windows och Mac) och öppnar pdf-dokumentet i detta program, kan man spela upp det animerat.
Om detta inte skulle funka kan man testa att installera Adobe Reader, Adobe Flashplayer och Adobe Air. Då ska pencast-pdf-dokumenten vara "spelbara" om man öppnar dem i Acrobat Reader.
Att kunna se pdf-dokumenten anmimerade innebär framför allt att man kan se i vilken ordning olika steg i lösningsresonemangen görs. Dessutom kan man höra min röst som ytterligare motiverar olika steg i lösningarna. Ofattbart torrt och träigt förvisso (kan inte förnekas) men de kan ändå tjäna som värdefull information för att komma underfund med hur man löser vissa typer av problem.
