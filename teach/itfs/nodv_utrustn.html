Nödvändig utrustning för att läsa Matematisk statistik
För att förvärva poäng från kursen Matematisk statistik krävs endast att man klarar examinationskraven (för närvarande att man skriver betyget Godkänt eller Väl godkänt på tentamen för kursen) och ingen teknisk utrustning alls. För att utnyttja den service som ges för att förbättra sina chanser till goda resultat kan man dock:
    		• Kommunicera via email
    Då behövs mail-läsare.
    		• Inhämta information via kurshemsidan
    Då behövs en internetbrowser, t.ex. Mozilla Firefox (tillgänglig gratis), Netscape (tillgänglig gratis), Internet Explorer, Safari eller Opera (demo-version tillgänglig gratis.)
    		• Läsa och ladda ned dokument från kurshemsidan
    Då behövs internetbrowser (se ovan) och pdf-läsare, t.ex. Acrobat reader (gratis) eller xpdf (gratis).
    		• Delta på distansföreläsningarna
    Då behövs internetbrowser (se ovan) med Javastöd. Java fås genom att besöka http://www.java.com/en, gå vidare till download, ladda ned den senaste versionen (gratis) och installera. Om man har Mozilla Firefox som internetbrowser bör man sedan man startat programmet välja EDIT -> PREFERENCES -> WEB FEATURES och kryssa för "Enable Java".
